## 📕 About

**EN** = This JavaScript code is a BMI calculator that uses modals to display results based on user inputs of height and weight. It provides descriptions and visual feedback according to the calculated BMI.

**PT** = Este código JavaScript é uma calculadora de IMC que usa modais para exibir resultados com base nos inputs de altura e peso do usuário. Oferece descrições e feedback visual de acordo com o IMC calculado.

## 🛠️ Features

- **Code Modularity:** The code is modular, making maintenance and future updates easier.
- **Responsive Design:** The project is adapted for mobile devices, providing an optimized user experience across all platforms.
- **Interactive Modal:** Results are displayed in interactive modals with personalized descriptions based on the calculated BMI.
- **Visual Feedback:** Images and colors are used to provide visual feedback based on the BMI result.
- Data Validation: The code includes validations to ensure the accuracy of input data.
